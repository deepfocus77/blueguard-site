BlueGuard Privacy Policy BlueGuard 개인정보처리방침

1. Overview

BlueGuard ("the Extension") is a Chrome extension designed to prevent sensitive data leakage when using AI services in enterprise environments. This Privacy Policy describes the data that BlueGuard collects, uses, and stores.

2. Data We Collect

BlueGuard processes the following types of data:

  • AI Service Input Text: Text entered by users into supported AI services (ChatGPT, Gemini, Claude, Grok, Copilot, Perplexity, Genspark) is scanned in real-time. The text itself is not stored; only the presence of pre-registered keywords is checked.
  • Attached File Metadata: Document files (PDF, DOCX, PPTX, XLSX) uploaded to AI services are scanned for sensitive content. Files are analyzed locally only and are not transmitted externally.
  • Security Event Logs: When a keyword block or warning occurs, the event type, timestamp, and detected keyword category are sent to the enterprise proxy server. The original text entered by the user is not included.

3. Purpose of Data Use

  • Prevention of sensitive information leakage in accordance with enterprise security policies
  • Monitoring AI service usage and security auditing
  • Security policy violation alerts and reporting

4. Data Storage and Transmission

  • Keyword scanning is performed locally in the user's browser only.
  • Security event logs are transmitted only to the enterprise's internal proxy server and are not sent to any third-party servers.
  • Personal data, browsing history, passwords, or other personal information are not collected.

5. Third-Party Sharing

BlueGuard does not sell, trade, or provide collected data to any third parties. All data is used exclusively for the enterprise's internal security management purposes.

6. User Rights

  • Users can check the current security status through the extension's popup.
  • All locally stored data is deleted when the extension is removed.
  • For requests to access security event logs, please contact your organization's security administrator.

7. Security

BlueGuard uses encrypted communication through the enterprise's internal network for data transmission. Local data is securely stored in Chrome's storage (chrome.storage).

8. Changes

This Privacy Policy may be updated as needed. Any changes will be posted on this page.

9. License

BlueGuard is not for sale and is freely available for use. This software is provided free of charge with no commercial purpose.

1. 개요

BlueGuard("본 확장 프로그램")는 기업 환경에서 AI 서비스 사용 시 민감 데이터 유출을 방지하기 위한 Chrome 확장 프로그램입니다. 본 개인정보처리방침은 BlueGuard가 수집, 사용, 보관하는 데이터에 대해 설명합니다.

2. 수집하는 데이터

BlueGuard는 다음과 같은 데이터를 처리합니다:

  • AI 서비스 입력 텍스트: 사용자가 지원 대상 AI 서비스(ChatGPT, Gemini, Claude, Grok, Copilot, Perplexity, Genspark)에 입력하는 텍스트를 실시간으로 검사합니다. 텍스트 자체를 저장하지 않으며, 사전 등록된 키워드 포함 여부만 확인합니다.
  • 첨부 파일 메타데이터: AI 서비스에 업로드되는 문서 파일(PDF, DOCX, PPTX, XLSX)의 내용을 검사합니다. 파일은 로컬에서만 분석되며 외부로 전송되지 않습니다.
  • 보안 이벤트 로그: 키워드 차단/경고 발생 시, 이벤트 유형, 발생 시간, 감지된 키워드 카테고리를 기업 프록시 서버로 전송합니다. 사용자가 입력한 원문 텍스트는 포함되지 않습니다.

3. 데이터 사용 목적

  • 기업 보안 정책에 따른 민감 정보 유출 방지
  • AI 서비스 사용 현황 모니터링 및 보안 감사
  • 보안 정책 위반 알림 및 리포팅

4. 데이터 저장 및 전송

  • 키워드 검사는 사용자의 로컬 브라우저에서만 수행됩니다.
  • 보안 이벤트 로그는 기업 내부 프록시 서버로만 전송되며, 외부 제3자 서버로 전송되지 않습니다.
  • 사용자의 개인 데이터, 브라우징 기록, 비밀번호 등은 수집하지 않습니다.

5. 제3자 제공

BlueGuard는 수집된 데이터를 제3자에게 판매, 교환, 또는 제공하지 않습니다. 모든 데이터는 해당 기업의 내부 보안 관리 목적으로만 사용됩니다.

6. 사용자 권리

  • 사용자는 확장 프로그램의 팝업을 통해 현재 보안 상태를 확인할 수 있습니다.
  • 확장 프로그램 제거 시 로컬에 저장된 모든 데이터가 삭제됩니다.
  • 보안 이벤트 로그에 대한 열람 요청은 소속 기업의 보안 관리자에게 문의하시기 바랍니다.

7. 보안

BlueGuard는 데이터 전송 시 기업 내부 네트워크를 통해 암호화된 통신을 사용하며, 로컬 데이터는 Chrome의 보안 저장소(chrome.storage)에 안전하게 보관됩니다.

8. 변경 사항

본 개인정보처리방침은 필요에 따라 업데이트될 수 있으며, 변경 시 본 페이지를 통해 고지합니다.

9. 라이선스

BlueGuard는 판매 목적이 아니며, 누구나 자유롭게 사용할 수 있습니다. 본 소프트웨어는 상업적 목적 없이 무료로 제공됩니다.
