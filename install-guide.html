BlueGuard 설치 가이드

Chrome 확장 프로그램 수동 설치 매뉴얼

안내: 이 가이드는 BlueGuard 확장 프로그램을 Chrome에 수동으로 설치하는 방법입니다.
소요 시간: 약 2~3분 | 난이도: 쉬움
1

zip 파일 압축 해제

배포받은 blueguard-extension.zip 파일을 바탕화면이나 원하는 폴더에 압축 해제합니다.

zip 파일을 마우스 오른쪽 클릭 → "압축 풀기" 또는 "여기에 풀기" 선택

📁 blueguard-extension
📄 manifest.json
📄 background.js
📄 content-script.js
📄 popup.html
📁 icons/
📁 lib/
📁 styles/
...
압축 해제 후 폴더 구조
주의: 압축 해제한 폴더를 삭제하면 확장 프로그램이 작동하지 않습니다. 삭제하지 않을 위치에 압축을 풀어주세요.
2

Chrome 확장 프로그램 페이지 열기

Chrome 브라우저를 열고, 주소창에 아래 주소를 입력합니다:

주소창에 입력: chrome://extensions → Enter

← →
chrome://extensions
확장 프로그램
Chrome 확장 프로그램 관리 페이지
3

개발자 모드 켜기

확장 프로그램 페이지의 오른쪽 상단에 있는 개발자 모드 토글을 켭니다.

확장 프로그램
개발자 모드

업데이트
개발자 모드를 켜면 상단에 3개의 버튼이 나타납니다
4

"압축해제된 확장 프로그램을 로드합니다" 클릭

개발자 모드를 켜면 나타나는 압축해제된 확장 프로그램을 로드합니다. 버튼을 클릭합니다.

업데이트
왼쪽 첫 번째 버튼을 클릭하세요
5

압축 해제한 폴더 선택

폴더 선택 창이 열리면, STEP 1에서 압축 해제한 폴더를 선택하고 폴더 선택을 클릭합니다.

중요: manifest.json 파일이 있는 폴더를 선택해야 합니다.

예시: C:\Users\사용자\Desktop\blueguard-extension

📁 blueguard-extension
폴더 선택
manifest.json이 들어있는 폴더를 선택합니다
6

설치 완료 확인

설치가 완료되면 확장 프로그램 목록에 BlueGuard - AI 보안 게이트웨이가 나타납니다.

7

도구 모음에 고정 (선택사항)

Chrome 주소창 오른쪽의 퍼즐 아이콘 (🔧)을 클릭한 후, BlueGuard 옆의 핀 아이콘을 클릭하면 도구 모음에 항상 표시됩니다.

⚠ 문제 해결

Q: "매니페스트 파일이 누락되었거나 읽을 수 없습니다" 오류가 나타나요

zip 파일 안의 폴더가 아닌, manifest.json이 직접 들어있는 폴더를 선택해야 합니다.
폴더를 열었을 때 manifest.json 파일이 바로 보여야 합니다.

Q: 확장 프로그램이 "사용 중지됨"으로 나타나요

확장 프로그램 카드의 토글 스위치를 클릭하여 사용으로 변경해주세요.

Q: Chrome 업데이트 후 확장 프로그램이 사라졌어요

STEP 2~5를 다시 수행하여 재설치해주세요. 압축 해제한 원본 폴더가 남아있어야 합니다.
