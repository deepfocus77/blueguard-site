BlueGuard

AI Security Gateway - A Chrome extension that monitors AI service usage and prevents sensitive data leakage in enterprise environments. AI 보안 게이트웨이 - 기업의 AI 서비스 사용을 모니터링하고 민감 데이터 유출을 방지하는 Chrome 확장 프로그램

Chrome Extension · Enterprise Security · Free & Open Use
🔒

Keyword-Based Protection 키워드 기반 보안

Detects pre-registered sensitive keywords in real-time and blocks or warns before data is submitted to AI services. 사전 등록된 키워드를 감지하여 민감 정보의 AI 서비스 입력을 실시간으로 차단하거나 경고합니다.

📄

Document File Scanning 문서 파일 스캔

Automatically scans attached files (PDF, DOCX, PPTX, XLSX) for sensitive data before upload to AI services. PDF, DOCX, PPTX, XLSX 등 첨부 파일을 자동 분석하여 민감 데이터 포함 여부를 검사합니다.

📡

Proxy-Based Monitoring 프록시 기반 모니터링

Centrally monitor AI service usage through an enterprise proxy server with comprehensive logging and reporting. 기업 프록시 서버를 통해 AI 서비스 사용 현황을 중앙에서 모니터링하고 로그를 관리합니다.

How It Works 동작 원리

1

Install 설치

Administrators deploy the EXE installer, and the Chrome extension is automatically installed. 관리자가 EXE 인스톨러를 배포하면 Chrome에 자동으로 확장이 설치됩니다.

2

Detect 감지

When users input text or upload files to AI services, keywords are scanned in real-time. 사용자가 AI 서비스에 텍스트 또는 파일을 입력하면 실시간으로 키워드를 검사합니다.

3

Block / Warn 차단 / 경고

If sensitive keywords are found, the input is blocked or a warning overlay is displayed. 민감 키워드 발견 시 입력을 차단하거나 경고 오버레이를 표시합니다.

4

Report 리포트

All activities are logged to the proxy server and available on the admin dashboard. 모든 활동은 프록시 서버에 로깅되어 관리자가 대시보드에서 확인합니다.
